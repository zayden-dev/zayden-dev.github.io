Mes experiences

Je vais d'abord vous parler ici de mes stages de première et deuxième année de BTS, ensuite je vous présenterai mes projets réalisés pendant ces années et enfin un projet personnel.

1ère année

Mon stage de première année a été réalisé auprès d'une entreprise d'évènementiel dans le jeu vidéo. En effet, Les Francs Gamers organise des tournois de jeu vidéo en partenariat avec les communes/régions. Durant mon stage fait en duo nous étions exclusivement à distance dû à un petit virus connu sous le doux nom de coronavirus SARS-COv2, ainsi nous étions en contact permanent en chat vocal et nous avons réalisé le site complet de l'entreprise. Ce site a été élaboré en mélange de code pur avec une intégration wordpress pour bénéficier de ses modules et certains de ses atouts. Il était d'ailleurs intéressant de communiquer en anglais avec une entreprise polonaise qui était auteure du module utilisé pour notre site qui comporte un système de création et gestion de tournoi, création d'équipe, gestion de compte/profil et pouvoir l'adapter au mieux à notre cahier des charges. Dû à un récent déménagement du chef d'entreprise, le site est actuellement éteint en attente de nouveau partenariat dans la nouvelle region.

2ème année

Présentation de l'entreprise : ATEXIOM est une société par action simplifiées (SAS) créée en juin 2015 dont le siège est situé à Lieusaint en Seine et Marne comprenant 2 salariés. L’entreprise a pour activité le Conseil en systèmes et logiciels informatique, dans le domaine des paiements et de la communication bancaire, des garanties bancaires (Trade Finance) ainsi que de la gestion d’actifs (immobilisations). ATEXIOM édite et propose également une solution innovante d’archivage applicatif.
- Au sein de l’entreprise Atexiom j'ai suivi une formation JavaScript et réalisation d'un système de clé de licence (cryptage et décryptage personnalisé). De plus, j'ai participé à la refonte du site web de l'entreprise. Formations réalisées en entreprise : approfondissement de mes connaissances en JavaScript, HTML, CSS et PHP approche du DOM organisation projet, utilisation de Filezilla(FTP), utilisation de phpmyadmin(DB). Tâches effectuées : création du système de cryptage de clé de licence refonte du site vitrine de l'entreprise.


Projets en cours de formation

Vetux-Line

Le Projet Vetux line a pour objectif de créer une application utilisant le framework PHP Symfony permettant de fusionner des fichiers csv. Il faut que l’utilisateur puisse choisir entre un mélange séquentiel ou entrelacé pour les fichiers csv. Aussi les données de cette fusion crée un fichier téléchargeable et peut également être directement modifiées sur l'application (ajouter supprimer une donnée etc...). Les tables dont on voulait créer une fusion pouvait être directement importée en fichier mais on pouvait également utiliser les tables issues d'une base de données.

SushiFast

Le projet SushiFast a pour objectif de proposer une application destinée à un intermédiaire entre le client et le restaurant de sushi lui permettant de passer des commandes, gérer les commandes, suivre les livraisons, l'historique et aussi pouvoir consulter son catalogue. Cette application web à été réalisée sous le framework Angular dans sa version en TypeScript.

CodeAlpha

Le projet CodeAlpha consiste à gérer les interventions des intervenants dans une centrale nucléaire, de générer et d'envoyer un QR Codes via une adresse mail sécurisé aux intervenants suite à leurs demandes. Ce projet utilise une structure sous nodeJS et une liaision permanente avec une API qui stock toutes les informations de passage.

Botanica

Le projet est une réalisation d'un site de e-commerce et vitrine complet, d'une entreprise fictive. Il agit comme réalisation front-end et back-end avec une interaction permanente avec une API Rest, React offre une SPA (Single Page Application) très intéressante et fluide.

Projets personnels

Lucky Noobs

Lors de certains temps libre, j'ai, avec des amis allemands, liés notre passion pour un jeu vidéo et le développement web. C'est un site codé en php/html/css, lié à un serveur où est installé l'outil wordpress. Dans notre situation, wordpress nous sert de base de données d'élements. Je m'explique, chaque page, chaque bloc est déjà prérécrit dans de nombreux fichiers que nous avons fait, ensuite il ne suffit plus qu'à modifier le texte ou les éléments API via wordpress, surtout que le site est proposé en 2 langues, en anglais et en allemand. Ce site répertorie de nombreux guides et vidéos sur le jeu vidéo Guild Wars 2 que je ne vais pas détailler ici.

D'ailleurs nous pensons déjà potentiellement à une refonte du style, dont le design a déjà été réalisé via Adobe XD (logiciel de conception de design).
